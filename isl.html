Project name

IMSUBEB E-Learning Official Website

Project Description

An E-learning Website For Schools in Imo State. As a result of the outbreak of the corona virus and with the new stay at home policy, We were tasked to provide a solution for learning for students in the state. Hence the design and development of this website. Equipped with different learning methods, online classes, live chats, materials downloads as well as so much more.

Tech Stack

  • CSS
  • Laravel(PHP)
  • Vue Js(JavaScript)
